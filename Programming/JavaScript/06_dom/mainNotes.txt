window : The window object in JavaScript is a global object that represents the window or tab in which
 your script is running. It serves as the global execution context (or global object) for all JavaScript code
 running in the browser.All global variables, functions, and objects are properties of the window object.

# Window object : Just type window in console it will display all global variables, functions, 
and objects are properties of the window object
